This project contains two methods to replace a particular character in a string at all occurences. 
1) iterateAndRemoveChar  //Iterate over whole String
2) oneLineRemoveChar    // Removes character with default ReplaceAll string method

Test Cases -
Both the methods are tests for - 
1) Alphabter
2) Number
3) Special Character
4) Empty Space send as character to be replaced
5) Null
